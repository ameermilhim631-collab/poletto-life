## 🎛️ لوحة التحكم (Admin Panel)

### تسجيل الدخول
```
اسم المستخدم: owner
كلمة المرور: pollitos4
```

### مميزات لوحة التحكم

#### إدارة المنتجات
- إضافة منتجات جديدة
- تعديل المنتجات الموجودة
- حذف المنتجات
- رفع صور للمنتجات
- تحديد الفئات والأنواع
- تفعيل/تعطيل المنتجات

#### إدارة الطلبات
- عرض جميع الطلبات
- تصفية الطلبات حسب الحالة
- تحديث حالة الطلب (قيد الانتظار/مكتمل/ملغى)

#### إدارة المستخدمين والصلاحيات
- إضافة مستخدمين جدد
- تعديل صلاحيات المستخدمين
- حذف المستخدمين

**الصلاحيات المتاحة:**
| الصلاحية | الوصف |
|----------|-------|
| **مدير (Admin)** | صلاحيات كاملة |
| **محرر (Editor)** | تعديل المنتجات والطلبات |
| **مشاهد (Viewer)** | عرض فقط |

**الصلاحيات التفصيلية:**
- ✅ إدارة المنتجات
- ✅ إدارة الطلبات
- ✅ إدارة المستخدمين
- ✅ الإعدادات
- ✅ عرض المتجر

#### الإعدادات
- تعديل اسم المتجر
- تعديل وصف المتجر
- تحديث رابط الديسكورد
- تحديث عنوان السيرفر
- تفعيل/تعطيل طرق الدفع
- تغيير كلمة المرور

#### تصدير البيانات
- تصدير جميع البيانات كملف JSON

---

## كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. سجل دخول بحسابك
3. تصفح المنتجات واختر ما يناسبك
4. أكمل عملية الشراء

### للدخول للوحة التحكم:
1. افتح ملف `admin.html`
2. سجل دخول بالبيانات التالية:
   -اسم المستخدم: `owner`
   - كلمة المرور: `pollitos4`
